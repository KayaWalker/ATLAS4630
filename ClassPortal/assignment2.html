Assignment 2

Create a web page with a form that does the following:

  1. Use at least one button tag or input type="submit".
  2. Use at least three input tags other than the button with different typesLinks to an external site.
  3. In addition, use at least one of the following other form controls: textarea, select and option, datalist and option, progress or meter
  4. There must be at least one form tag in the page that is a parent to all of the form control tags.
  5. The page should change based on the form input.
  6. Event response must use event listeners. Javascript and CSS must be linked in external files.
